 Voice Assistant (Python)

A custom Python-based Voice Assistant that can listen to your voice commands, respond with speech, and automate daily tasks like opening applications, searching the web, telling time/date, and more.

This project is beginner‑friendly and helps you understand Speech Recognition, Text‑to‑Speech, and Automation using Python.

Features

 Voice command recognition

 Text‑to‑Speech responses

 Google / YouTube search

 Current time & date

Open system applications
